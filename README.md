## 📚 Sobre o Projeto

Este repositório contém a implementação dos exercícios da disciplina de Algoritmos e Lógica de Programação do primeiro semestre da FATEC. O projeto é desenvolvido em C e apresenta uma interface de linha de comando interativa para execução dos exercícios.

## 🌟 Características

- Interface de usuário com menu interativo
- Implementação modular dos exercícios
- Suporte multiplataforma (Windows/Linux)
- Banner ASCII art personalizado
- Estrutura organizada por exercícios

## 🔧 Pré-requisitos

- Compilador C (GCC recomendado)
- Make (opcional, para automatizar a compilação)
- Terminal com suporte a caracteres UTF-8

## 📥 Como Usar

1. Clone o repositório:
```sh
git clone https://github.com/oprimogus/dsa-fatec1/.git
```

2. Navegue até o diretório do projeto:
```sh
cd dsa-fatec1
```

3. Execute o programa:
```sh
make
```
